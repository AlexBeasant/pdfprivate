Privacy Policy

Last updated: April 14, 2026

PDF Private does not access, receive, store, or transmit any files you process. All PDF processing happens entirely within your web browser on your own device. Your documents never leave your computer.

1. How PDF Private Works

PDF Private provides browser-based tools for processing PDF files. When you use any tool on this site, the following is true:

  • Your files are loaded into your browser's local memory only.
  • All processing is performed by JavaScript and WebAssembly code running on your device.
  • No file data is uploaded to any server at any time.
  • No file contents are transmitted over the internet.
  • When you close the browser tab, all file data in memory is discarded by your browser.

PDF Private has no server-side file processing infrastructure. There is no upload endpoint, no file storage, and no mechanism by which we could access your documents even if we wanted to.

2. What We Do Not Collect

We do not collect, store, access, or process:

  • Any PDF files or their contents
  • Any images, text, or data extracted from your files
  • File names, file sizes, or file metadata
  • Personal information such as your name, email address, or account credentials (no accounts exist)
  • Keystroke data, form inputs, or user-generated content of any kind

3. What We May Collect

To understand how visitors find and use the website (not the files you process), we may use standard web analytics services such as Google Analytics or similar tools. These services may collect:

  • Anonymised page view data (which pages are visited, how long visitors stay)
  • Referral sources (how you found the site, e.g. search engine, direct link)
  • General geographic region (country or city level, not precise location)
  • Browser type and device category (e.g. mobile, desktop)

This data is aggregated, anonymised, and used solely to understand traffic patterns and improve the website. It contains no information about the files you process and cannot be used to identify you personally.

4. Cookies

PDF Private itself does not set any cookies. If we use a third-party analytics service, that service may set its own cookies in accordance with its own privacy policy. You can block these cookies using your browser settings or a browser extension without affecting the functionality of any PDF tool on this site.

5. Third-Party Services

The website may use the following types of third-party services:

  • Content Delivery Networks (CDNs) — to serve JavaScript libraries (such as PDF.js and pdf-lib) that power the client-side tools. These CDNs may log standard access data (IP address, timestamp) as part of their normal operations, in accordance with their own privacy policies.
  • Analytics services — as described in Section 3 above.
  • GitHub Pages — the site is hosted on GitHub Pages. GitHub's privacy practices are governed by GitHub's privacy statement.

None of these third-party services have any access to the files you process using PDF Private's tools.

6. Data Security

Because your files are processed entirely on your own device, the security of your documents depends on the security of your own device, browser, and network environment. PDF Private does not control and is not responsible for the security of your local environment, including any browser extensions, malware, or other software installed on your device.

7. Children's Privacy

PDF Private does not knowingly collect any personal information from anyone, including children under the age of 16. Since no accounts or personal data are collected, there is no age-related data processing.

8. Your Rights

Because we do not collect personal data through the use of our PDF tools, there is no personal data to access, correct, delete, or port. If you have questions about the anonymised analytics data described in Section 3, you may contact us using the details below.

9. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated revision date. Your continued use of the site after changes are posted constitutes acceptance of the updated policy.

10. Contact

If you have any questions about this Privacy Policy, please contact us at: privacy@pdfprivate.com
