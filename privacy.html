Privacy Policy

Last updated: January 2025

The Short Version

Your files never leave your device. All PDF processing happens entirely in your web browser using JavaScript. We don't upload, store, or have access to any of your files.

No File Uploads

PDF Private is designed with privacy as its core principle. When you select a file to process:

  • The file is read directly by your web browser
  • All processing happens locally on your device
  • The file is never uploaded to any server
  • We have no access to your files or their contents
  • Once you close the browser tab, all data is gone

No Cookies

We do not use cookies. There are no tracking cookies, analytics cookies, or any other type of cookies stored on your device when using our tools.

No Personal Data Collection

We do not collect any personal information. Specifically:

  • We don't ask for your name, email, or any contact information
  • We don't track your IP address
  • We don't use analytics services
  • We don't create user profiles or accounts
  • We don't share any data with third parties (because we don't have any data to share)

No Data Storage

Your files and any processed outputs exist only temporarily in your browser's memory while you're using the tool. We do not:

  • Store your files on any server
  • Create backups of your files
  • Cache your files anywhere
  • Transmit your files over the internet

Third-Party Libraries

We use open-source JavaScript libraries to process PDFs and images. These libraries run entirely in your browser and do not transmit any data:

  • pdf-lib - For PDF manipulation
  • PDF.js - For rendering PDF pages
  • jsPDF - For creating PDFs
  • JSZip - For creating ZIP archives

Changes to This Policy

If we make any changes to this privacy policy, we will update the "Last updated" date at the top of this page. We encourage you to review this policy periodically.

Questions?

If you have any questions about this privacy policy or our privacy practices, please feel free to contact us.

All processing happens in your browser. Your files never leave your device.
